# 🌌 Artelligence OS – Unified Digital Workspace & Agent Orchestrator

تطبيق **Artelligence OS** هو لوحة تحكم سطح مكتب فائقة التطور، مخصصة لإدارة وتنسيق وكلاء الذكاء الاصطناعي وتتبع إنتاجية المطورين محلياً بالكامل وبخصوصية 100%. التطبيق الآن مستقل 100% ويعمل بدون أي اعتماديات خارجية أو بايثون!

---

## 🚀 دليل التثبيت والتشغيل (Step-by-Step Installation)

### 📥 الخطوة 1: تثبيت التطبيق (Installation)
1. قم بفتح مجلد توزيع التطبيق: `dist/`
2. اضغط مرتين على ملف التثبيت **`Artelligence OS-1.0.0-arm64.dmg`**.
3. اسحب أيقونة تطبيق **Artelligence OS** إلى مجلد التطبيقات **Applications**.

### 🛡️ الخطوة 2: تخطي فحص الأمان لنظام macOS (Gatekeeper Bypass)
بما أن التطبيق مفتوح المصدر وموزع مجاناً خارج متجر تطبيقات آبل الرسمي وبدون شهادة مدفوعة، سيقوم النظام بحظر تشغيله تلقائياً ويعرض رسالة تنبيه. لتخطي هذا القيد بأمان تام:

#### 🖱️ الطريقة السهلة (للمستخدمين):
1. افتح مجلد التطبيقات **Applications** في Finder.
2. اضغط بزر الفأرة الأيمن **(Right-Click)** أو اضغط مع زر **Control** على أيقونة **Artelligence OS**.
3. اختر **Open** من القائمة المنبثقة.
4. ستظهر لك نافذة تأكيد، اضغط على **Open** لتشغيل التطبيق فوراً. (تحتاج لفعل هذا في المرة الأولى فقط).

#### 💻 طريقة المطورين (عبر Terminal):
افتح نافذة Terminal وقم بتشغيل الأمر التالي لإزالة قيود الحجر الصحي (Quarantine Attributes) عن التطبيق نهائياً:
```bash
xattr -cr "/Applications/Artelligence OS.app"
```

## 🛠️ الميزات المدمجة بالتطبيق (Core Features)
1. **لوحة القيادة الذكية (Dashboard):** تعرض إحصائيات النشاط الفعلي وساعات التركيز محلياً بتحديث كل 5 ثوانٍ.
2. **مخطط اتصالات الوكلاء (Agent Connect Graph):** شبكة رسومية تفاعلية تعرض نبضات نقل البيانات وتكامل الوكلاء.
3. **أيقونة شريط القوائم (macOS Menu Bar Icon):** للتحكم السريع وإظهار/إخفاء التطبيق من شريط المهام العلوي.
4. **الاختصار العالمي (Global Hotkey):** اضغط على `Cmd+Ctrl+A` لفتح وإخفاء التطبيق فوراً من أي مكان بجهازك.
5. **منظم المجلدات التلقائي (Workspace Organizer):** فرز وتصنيف ملفات مشاريعك في مجلد التنزيلات وتوجيهها للمجلد المناسب تلقائياً.

## 📝 متطلبات النظام (System Requirements)
* جهاز كمبيوتر يعمل بنظام تشغيل macOS 10.12+ (معالجات Apple Silicon M1/M2/M3).
* لا يتطلب بايثون ولا أي إعدادات مسبقة - تشغيل فوري!
